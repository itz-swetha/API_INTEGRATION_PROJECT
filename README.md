## API INTEGRATION PROJECT
# Country Information App

This is a simple frontend web application that fetches country data from a public REST API and displays it dynamically on a webpage.

## Technologies Used
- HTML
- CSS
- JavaScript (Fetch API)

## API Used
REST Countries API  
https://restcountries.com/

## Features
- Fetches live country data
- Displays country flag, name, capital, region, and population
- Search functionality to filter countries
- Responsive layout with multiple cards per row

## How to Run
Open the project folder in VS Code and run `index.html` using Live Server.
